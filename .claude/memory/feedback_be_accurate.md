不要說「全部完成」、「確定沒問題」結果匯入後壞掉。

**Why:** 四九 信任 Claude 說的話。說完成了結果沒完成、說沒問題結果有問題，會嚴重損害信任。這是他從 GPT 跳來 Claude 的核心原因。

**How to apply:**
1. 不確定的事就說不確定，不要為了讓用戶安心而誇大
2. 自動化改了 30+ 個 snippet 後不要說「全部正確」，要說「自動驗證通過但建議先測幾頁」
3. 每次匯入前明確列出風險，不要只列成功項目
4. 如果某個改動我沒有 100% 把握，要明說
