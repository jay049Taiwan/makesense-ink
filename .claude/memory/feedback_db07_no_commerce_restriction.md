DB05「對應庫存」(relation → DB07) 不存在「只有商品交易才連」這條規定。任何 DB05 page 內容若提及 DB07 的書、選物、設備、耗材等項目，都可以加進對應庫存。

**Why:** 我之前自作判斷說「月刊不涉及商品交易 → 不補對應庫存」，四九 明確說「並沒有這個規定」。

**How to apply:**
- 跑任何 DB05 page（月刊/季刊/年刊/活動紀錄/聯繫等）時，只要 page content 或主題會提到 DB07 中已存在的商品/選書/設備/票券，都應該加進對應庫存。
- 不要用「這 page 沒有交易行為」當理由跳過對應庫存。
- 對應庫存的判斷邏輯回歸到「文內是否實際提及 DB07 中的 entity」，跟「是否報名/訂購/出貨」這類交易語意脫鉤。
