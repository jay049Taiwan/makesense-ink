## What
寫進 memory(記憶檔) 的內容、以及給四九的一般說明，一律遵守兩條：

1. **寫全稱、不用內部黑話簡稱**。
   - 例：不要寫「棒」，要寫全稱「DB06 分析記錄(DB06 analysis record)」。
   - 原因：同一個簡稱常被拿去套不同狀況（「棒」一下指交接棒、一下指分析記錄），非常容易混淆。

2. **英文詞後面一定括號中文**，格式 `English(中文)`。
   - 例：`dog(狗)`、`baton(交接棒)`、`relation(關聯欄位)`、`session(對話階段)`、`token(語素)`。

## Why
四九 反映：我(Claude) 愛用「棒」這種簡稱，而且同一個字套不同情境，非常容易搞混。全稱 + 英文附中文能消除歧義。

## How to apply
- 任何要寫進 memory(記憶檔) 的檔，動筆前自我檢查：有沒有黑話簡稱？有沒有裸英文沒附中文？
- 一般對話回覆也比照辦理：盡量全稱、英文一律附中文。
