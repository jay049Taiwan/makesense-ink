# 公司品牌命名

- **對外品牌**：makesense（全名 **sense makes sense**，簡稱 **sms**）
- **中文法人名**：現思文化創藝有限公司（僅供正式登記/法律文件使用）
- **避免**：在 skill 名稱、檔案命名、對話用語使用 `xiansai` 或「現思」當主品牌
- **優先順序**：makesense > sms > 現思（中文場景）

**Why**：四九 明確指正過——習慣稱公司為 makesense 或 sms，xiansai 拼音對外不識別也不喜歡。

**How to apply**：
- 命名 skill / 檔案 / 變數時用 `makesense-*` 或 `sms-*`
- 既有 `n8n-makesense` 已是正確命名範例
- 講中文時用「makesense」或具體品牌（旅人書店、moku），不要拼成「現思」當主稱
- 寫到「現思文化」只在指法人實體時用
