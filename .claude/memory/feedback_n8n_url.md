提到任何 n8n workflow 時,一律直接給可點的完整 URL（`https://makesense.zeabur.app/workflow/<ID>`），不要只丟 workflow ID 讓四九自己組。

**Why:** 四九 每次都要再問一次連結,來回浪費時間。
**How to apply:** 任何回覆只要出現 n8n workflow ID,就同時附上 `https://makesense.zeabur.app/workflow/<ID>` 完整連結。
