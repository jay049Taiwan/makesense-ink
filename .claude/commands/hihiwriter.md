請依使用者輸入的 $ARGUMENTS 執行嗨嗨文案工作。

## 第一步：必讀工作指示

執行前先 fetch Notion 工作指示頁（https://www.notion.so/3639ff25fdab811eb356cc0b8a40e35f），
嚴格按其「動筆前確認流程」→「動筆」→「完成回寫鐵律」三段執行。

## 動筆前確認流程（工作指示頁的簡化版，完整版以 Notion 為準）

1. **讀執行構想**：target page 的「執行構想」欄位有內容且嗨嗨企劃已補充章節架構
2. **讀欄位組合指南**（https://www.notion.so/35e9ff25fdab80269dc4f65a7ce83305）：找到 target 的身份名 → 對應子類撰寫指南 URL
3. **讀子類撰寫指南**：確認主體 / 緣由 / 對象 / 格式 / 架構 / 限制 + voice profile 指定
4. **讀 voice profile**：先讀共用（https://www.notion.so/35f9ff25fdab8134bd20e9fd5b5bb7fb），再讀子類指定 profile
5. **素材齊全確認**：執行構想 + 企劃筆記 + 必要 relation 資料都到位
6. 任一步驟缺料 → **停下，回報缺什麼，不強寫**

## 完成回寫鐵律

寫完後必做兩件事：
1. self-check 通過（對照子類指南的自檢清單）
2. 更新 target page 的「執行進度」→ 已完成；「智動狀態」→ 已完成（若為工項型觸發）

唯一可動 target page content，未通過 self-check 不設完成。
